# CryptoFlop

Osobiste narzędzie do codziennej obserwacji sentymentu rynku krypto (BTC, BNB).
Statyczny dashboard generowany 3× dziennie — pomaga w intuicji o nastroju rynku,
nie służy do predykcji cen.

Stack: HTML + CSS + vanilla JS (na razie mockup z dummy data).

Patrz `CLAUDE.md` dla pełnego kontekstu projektu i `colors.md` dla palety.
